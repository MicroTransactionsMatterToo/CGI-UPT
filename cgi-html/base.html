What is Kohknowco VM

Kohknowco VM is a system constructer by Ennis Massey to provide a light weight LXC wrapper that allows for easy creation, management and deletion of Linux containers. Similar to Docker, this system will allow users to create containers in which servers, software and other stuff can be hosted. This system allow for the user to have free reign over their machine without affecting others files/systems.

Who can use it?

This system is primarily designed for the use of students of Ao Tawhiti Unlimited Discovery School. As such, initially, only students of this school will be able to use it. I have however made plans to create an installation script, so it can be setup on any machine at all.


Setup

To get started with Kohknowco, simply fill in the required field below and submit them. Your Linux will then be created, and the ip will be displayed, allowing you to ssh into and configure it.
